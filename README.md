# Canvas Resize

## canvas.getContext("2d:)

Canvas 요소 자체는 그래픽을 표시할 영역을 뿐이다. 실제로 선, 도형, 텍스트, 이미지 등을 그리려면 context가 필요하다.

- getContext(): Canvas 요소에서 그리기 컨텍스트를 가져오는 메서드
- "2d": 2차원 그래픽 컨텍스트를 요청하는 매개변수
- 반환값: 2D 그래픽을 그릴 수 있는 모든 메서드와 속성을 가진 객체
